# 仙剑肉鸽 - 美术需求之角色设计

---

## 角色表现

### 主角形象
- 圆形底座（直径60px）
- 圆形内显示角色头像
- 头顶显示角色名称

### 血条设计
- 圆形进度条，围绕在角色圆形外圈
- 血量百分比显示：
  - 100%-51%：绿色
  - 50%-21%：黄色
  - 20%-0%：红色

### 小怪形象
- 与主角相同的表现形式
- 圆形底座 + 头像
- 圆形外圈血条

---

## 视觉效果示意

```
┌────────────────────────────┐
│                            │
│   [李逍遥]    [赵灵儿]     │  ← 80%血(绿)   45%血(黄)
│     ○           ○          │  ← 蓝色底座    蓝色底座
│     ↑           ↑          │  ← 外圈血条    外圈血条
│                            │
│   [小怪]      [小怪]       │  ← 15%血(红)   90%血(绿)
│     ○           ○          │  ← 红色底座    红色底座
│                            │
└────────────────────────────┘
```

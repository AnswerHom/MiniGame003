# 技能自动释放

## 需求概述

移除战斗HUD下方的技能快捷栏，改为自动释放技能。

## 需求内容

### 1. 移除技能快捷栏

- 删除战斗HUD下方的技能快捷栏UI
- 释放底部UI空间

### 2. 技能自动释放

- 战斗中技能自动释放，无需手动点击
- 技能自动释放间隔基于角色攻击速度换算
  - 例如：攻击速度1.0 = 每1秒释放一次技能
  - 攻速2.0 = 每0.5秒释放一次
  - 攻速0.5 = 每2秒释放一次
- 技能按技能栏顺序依次释放（从左到右）
- 大招单独计算冷却和释放间隔

### 3. 释放优先级

- 普通技能按顺序循环释放
- 大招独立判断冷却，冷却完成后自动释放

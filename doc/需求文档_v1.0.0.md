# 仙剑肉鸽 - 需求文档 v1.0.0

---

## 项目定位

- **类型**：肉鸽割草 + 组队生存
- **视角**：俯视上帝视角
- **平台**：手机端（竖屏）
- **特色**：仙剑IP角色 + 肉鸽卡牌Build

---

## 核心玩法

1. 5人角色组队协同战斗
2. 多波次怪物进攻，生存挑战
3. 角色全部阵亡 = 游戏结束
4. 大地图探索，虚拟摇杆控制移动
5. 抽卡选角 + 肉鸽随机事件

---

## 角色定位

剑客、巫师、药师

---

## 角色属性（7个）

生命值、攻击力、攻击速度、移动速度、攻击范围、暴击率、暴击伤害

---

## 角色设计

### 李逍遥（剑客）

**属性**
| 属性 | 数值 |
|------|------|
| 生命值 | 800 |
| 攻击力 | 120 |
| 攻击速度 | 1.0 |
| 移动速度 | 85 |
| 攻击范围 | 80px |
| 暴击率 | 10% |
| 暴击伤害 | 150% |

**技能**
| 技能名称 | 类型 | 效果 | 初始属性 |
|----------|------|------|----------|
| 御剑术 | 普通 | 向前方发射一道直线剑气 | 伤害150%、范围200px、冷却1.5秒 |
| 四方剑阵 | 普通 | 向四个方向发射四道剑气 | 剑数4、伤害80%×4、范围120px、冷却3秒 |
| 万剑护体 | 大招 | 剑环绕旋转，敌靠近受伤 | 伤害50%/秒、范围100px、持续10秒、冷却30秒 |

**肉鸽卡牌（12张）**
- 御剑术：锋芒毕露、分光化影、定身咒、万剑齐发
- 四方剑阵：剑荡八荒、剑阵加强、迟滞之阵、幻剑生花
- 万剑护体：剑锋凌厉、剑域扩张、剑鸣不绝、剑气回天

### 赵灵儿（药师）

**技能**
| 技能名称 | 类型 | 效果 | 初始属性 |
|----------|------|------|----------|
| 五雷咒 | 普通 | 治疗范围内队友 | 治疗80%攻击、范围150px、冷却2秒 |
| 观音咒 | 普通 | 为队友套护盾 | 护盾20%生命、范围100px、冷却4秒 |
| 圣灵复活 | 大招 | 复活死亡队友 | 复活1人、30秒冷却 |

**肉鸽卡牌（12张）**
- 五雷咒：雷光普照、群体治愈、持续治疗、雷劫
- 观音咒：金刚护体、护盾强化、反射护盾、水灵护盾
- 圣灵复活：复活祝福、群体复活、快速复活、复活传承

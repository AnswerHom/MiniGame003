# 需求文档_v2.32.0

## 需求概述

阿奴角色肉鸽卡牌设计：为阿奴的三个技能设计专属肉鸽卡牌

---

## 1. 风雪冰天 肉鸽卡牌（4张）

| 卡牌名称 | 效果 |
|----------|------|
| 冰魄银光 | 伤害+20% |
| 寒冰领域 | 冰锥射程+30px |
| 冰封千里 | 扇形角度60°→90° |
| 冰雪加身 | 攻击速度+15% |

---

## 2. 雷劫 肉鸽卡牌（4张）

| 卡牌名称 | 效果 |
|----------|------|
| 天雷滚滚 | 伤害+30% |
| 雷神之怒 | 雷电半径+20px |
| 麻痹之雷 | 命中目标定身0.5秒 |
| 引雷诀 | 雷电可穿透1个目标 |

---

## 3. 阴阳逆转 肉鸽卡牌（4张）

| 卡牌名称 | 效果 |
|----------|------|
| 阴阳相生 | 伤害+30% |
| 气震山河 | 弹开半径+30px |
| 护体神光 | 弹开时获得护盾（吸收30%最大生命值，持续3秒） |
| 阴阳轮回 | 大招冷却-10秒 |

---

## 4. 通用效果说明

- 所有加成效果为乘法叠加
- 同类型卡牌效果不可重复装备
- 装备界面显示卡牌效果预览

# 技能抽取机制

## 需求概述

调整角色技能获取方式，大招和部分技能需通过肉鸽卡池抽取。

## 需求内容

### 1. 初始技能配置

- 角色初始只携带普通技能1
- 普通技能2需要在肉鸽卡池抽取
- 大招需要在肉鸽卡池抽取

### 2. 技能抽取

- 肉鸽卡池中包含各角色的普通技能2和主动技能
- 抽到技能后自动装备到角色技能栏

### 3. 卡牌解锁机制

- 普通技能2的强化卡：只有在抽到普通技能2后，才会在卡池中随机到
- 大招的强化卡：只有在抽到大招后，才会在卡池中随机到
- 已解锁的技能，其强化卡才能被随到

### 4. 抽卡界面调整

- 卡池分类：初始技能包、进阶技能包（包含技能2和大招）
- 或者在原有卡池中区分已解锁/未解锁的技能卡

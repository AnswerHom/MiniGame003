# 角色攻击策略优化

## 需求概述

优化角色在战斗中的攻击策略，超出攻击范围时自动移动靠近目标后再攻击。

## 需求内容

### 1. 范围检测

- 每次技能即将释放时检测目标距离
- 角色攻击范围：各角色属性中的攻击范围值

### 2. 移动靠近逻辑

- 目标超出攻击范围时，角色自动向目标移动
- 移动至攻击范围内时停止移动
- 停止移动后立即释放技能

### 3. 优先级

- 优先检测当前选中的目标（如普攻/技能指向的目标）
- 无选中目标时，自动选择最近的目标

### 4. 移动表现

- 使用角色正常移动速度
- 移动过程中不释放技能（靠近后释放）
- 被打断时停止移动

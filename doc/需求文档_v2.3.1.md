# 需求文档_v2.3.1

## 队伍系统

### 队伍配置

- **最大人数**：5人
- **最少人数**：1人
- **入口**：游戏大厅→队伍配置按钮

### 队伍编辑界面

- **角色选择**：从已拥有的角色卡牌中选择
- **角色排序**：拖拽调整队伍顺序
- **角色展示**：已选角色显示在大厅底部

### 队形系统

- **队形类型**：
  - 横排：角色一字排开
  - 纵列：角色排队跟随
- **默认队形**：纵列
- **切换方式**：战斗前在大厅切换

### 战斗中的队伍行为

- **跟随移动**：第2-5号角色跟随第1号角色移动
- **跟随距离**：相邻角色间距30px
- **转向同步**：所有角色同步转向

# 需求文档_v2.0.0

## 阿奴（巫师）角色

### 角色基础属性

| 属性 | 数值 |
|------|------|
| 生命值 | 600 |
| 攻击力 | 150 |
| 攻击速度 | 0.8 |
| 移动速度 | 75 |
| 攻击范围 | 120px |
| 暴击率 | 15% |
| 暴击伤害 | 160% |

### 角色外观

- 基础形态：圆形角色，直径40px
- 头像：圆形位于角色中央，直径30px
- 血量环：圆形进度条环绕角色外圈，宽度3px
- 血量颜色：
  - 100%-51%：绿色 (#00FF00)
  - 50%-21%：黄色 (#FFFF00)
  - 20%-0%：红色 (#FF0000)

## 技能系统

### 技能1：风雪冰天

- **类型**：主动攻击技能
- **效果**：向前方扇形区域发射冰锥
- **关键参数**：
  - 伤害：120%攻击力
  - 扇形角度：60°
  - 射程：150px
  - 冰锥数量：5根
  - 飞行速度：300px/秒
  - 冷却时间：2秒
- **按键**：数字键1
- **特效**：蓝色冰锥 + 命中后冰冻减速效果（减速30%，持续1秒）

### 技能2：雷劫

- **类型**：主动攻击技能
- **效果**：指定区域降下雷电
- **关键参数**：
  - 伤害：180%攻击力
  - 半径：60px
  - 延迟：0.3秒后降落
  - 冷却时间：3秒
- **按键**：数字键2
- **特效**：黄色雷电下落 + 落地爆炸

### 技能3：阴阳逆转

- **类型**：主动大招
- **效果**：范围内的敌人被弹开并受到伤害
- **关键参数**：
  - 伤害：100%攻击力
  - 半径：150px
  - 弹开距离：100px
  - 持续时间：0.5秒（弹飞过程）
  - 冷却时间：25秒
- **按键**：数字键3
- **特效**：黑白阴阳气流旋转扩散

---

## 技能通用规则

- 技能图标：30x30px，正方形底板
- 冷却显示：技能图标上覆盖灰色遮罩，遮罩高度随冷却时间减少
- 快捷键：1/2/3分别对应技能1/2/3
- 施放方式：按下对应数字键即可释放
- 技能释放方向：自动朝向当前面向方向或最近敌人

---

## 战斗系统

### 普通攻击

- 攻击方式：远程魔法弹
- 攻击动画：角色前方发射魔法弹
- 攻击间隔：1/攻击速度 = 1.25秒
- 攻击范围：120px

### 暴击判定

- 暴击伤害：160%普通伤害
- 暴击特效：攻击时显示紫色闪光

---

## 移动控制

- 虚拟摇杆：屏幕左侧区域
- 摇杆大小：100px直径
- 摇杆样式：半透明圆形
- 移动响应：平滑移动

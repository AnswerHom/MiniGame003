# 需求文档_v2.0.0

## BOSS战系统

### BOSS触发规则

- **触发条件**：每5波触发1次BOSS战
- **波次提示**：第5/10/15/20...波显示"BOSS即将来袭"
- **战斗场景**：BOSS战后进入专属战斗区域

### BOSS类型

#### 毒娘子（第一形态）

- **生命值**：5000
- **攻击力**：80
- **攻击方式**：毒液喷射（前方扇形范围）
- **特殊能力**：
  - 每10秒召唤3只小怪
  - 血量低于30%时进入狂暴状态（攻速+50%）
- **弱点**：头部（攻击头部伤害+30%）

#### 赤鬼王（第二形态）

- **生命值**：8000
- **攻击力**：120
- **攻击方式**：
  - 地面冲击（圆形范围）
  - 岩石投掷（远程单体）
- **特殊能力**：
  - 每15秒释放地震（全屏伤害）
  - 免疫正面伤害（需绕后攻击）
- **弱点**：背部（攻击背部伤害+50%）

#### 拜月教主（最终形态）

- **生命值**：15000
- **攻击力**：150
- **攻击方式**：
  - 魔法弹（多发追踪）
  - 暗黑漩涡（持续范围吸附）
  - 召唤雷电（随机劈向角色）
- **特殊能力**：
  - 每20秒无敌3秒
  - 召唤小怪协助作战
- **弱点**：胸口水晶（攻击水晶伤害+30%）

### BOSS战斗机制

- **战斗时长**：无限制，直到击杀BOSS
- **复活限制**：BOSS战期间无法使用复活道具
- **团队配合**：需要至少2个角色存活才能继续战斗
- **伤害统计**：实时显示全队总伤害

### BOSS掉落

- **首杀奖励**：首杀BOSS获得500钻石
- **常规掉落**：
  - 金币：200-500
  - 随机卡牌：1-2张
  - 稀有道具：概率掉落

### BOSSUI

- **血条显示**：屏幕顶部显示BOSS名称和血条
- **伤害统计**：屏幕右上角显示团队总伤害
- **阶段提示**：BOSS进入新阶段时显示提示文字
- **战斗结束**：显示伤害统计和掉落物品

# 需求文档_v2.4.0

## 怪物美术与体型设计

### 角色体型

- **主角体型**：标准体型，半径20px
- **定位**：比小怪稍大，战斗时清晰可见

### 怪物体型

| 怪物等级 | 体型倍数 | 半径 |
|----------|----------|------|
| 小怪 | 1.0倍 | 15-18px（按类型） |
| 精英怪 | 1.3倍 | 20-24px |
| BOSS | 1.5倍 | 30px |

### 怪物外观

- **基础样式**：圆圈 + 圆形血条（与角色一致）
- **区分方式**：圆圈中间显示文字
  - 阴魂 → "阴"
  - 蝙蝠 → "蝠"
  - 毒蛇 → "蛇"
  - 蛤蟆 → "蛤"
  - 毒蜂 → "蜂"
  - 骷髅 → "骨"
  - 僵尸 → "僵"
  - 狐狸 → "狐"

### 怪物颜色（按类型）

- 坦克型（蛤蟆、僵尸）：深绿色 #2d5016 / #276749
- 敏捷型（毒蜂）：黄色 #d69e2e
- 战士型（骷髅）：白色 #e2e8f0
- 法师型（狐狸）：橙色 #ed8936
- 普通小怪：灰色 #a0aec0

### BOSS特效

- BOSS圆圈外加光效环绕（旋转光环）
- 光效颜色：金色 #ffd700
- 光效宽度：3px

# 需求文档_v2.3.0

## 战斗地图调整

### 地图尺寸

- **地图大小**：比当前地图大50%
- **视野范围**：根据屏幕尺寸自适应
- **地图边界**：可视化的地图边缘（有障碍物阻挡）

### 摄像机系统

- **跟随目标**：始终跟随队伍中的主导角色（一般是玩家操作的第一个角色）
- **跟随方式**：延迟跟随
- **延迟参数**：
  - 跟随延迟：0.3秒
  - 平滑系数：0.1
- **边界处理**：摄像机移动到地图边缘时停止

### 障碍物系统

- **障碍物类型**：
  - 岩石：大块石头，角色和敌人无法穿过
  - 树木：小树，角色和敌人无法穿过
  - 花丛：装饰物，可穿过
- **障碍物分布**：随机分布在地图上
- **障碍物效果**：
  - 阻挡角色和敌人移动
  - 敌人会绕过障碍物追击角色
- **视觉表现**：深色阴影区分障碍物区域

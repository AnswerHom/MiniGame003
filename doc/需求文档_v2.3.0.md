# 需求文档_v2.3.0

## 宠物系统

### 宠物类型

| 宠物 | 品质 | 技能 | 冷却 |
|------|------|------|------|
|  小石头 | 普通 | 嘲讽（吸引仇恨） | 8秒 |
|  小狐妖 | 稀有 | 魅惑（使敌人混乱） | 12秒 |
|  蝴蝶仙子 | 稀有 | 治疗（每秒回复2%生命） | 0秒 |
|  雷灵 | 史诗 | 雷电（范围伤害） | 15秒 |
|  火凤凰 | 传说 | 灼烧（持续伤害） | 20秒 |

### 宠物获取

- **抽卡附带**：抽角色卡时概率附带宠物
- **商店购买**：钻石商店随机刷新宠物
- **活动奖励**：限时活动赠送

### 宠物养成

- **升级消耗**：金币
- **升级效果**：每级+5%属性
- **进化条件**：达到满级后消耗同名宠物进化
- **进化效果**：进化后属性翻倍

### 宠物参战

- **上阵数量**：同时上阵1只宠物
- **跟随效果**：宠物自动战斗
- **切换功能**：可自由切换已拥有的宠物

### 宠物UI

- **入口**：主界面"宠物"按钮
- **列表**：显示已拥有的宠物
- **详情**：点击宠物显示属性和技能
- **上阵按钮**：选择宠物上阵
- **升级按钮**：消耗金币升级

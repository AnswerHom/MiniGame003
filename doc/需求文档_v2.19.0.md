# 李逍遥技能调整

## 需求概述

调整李逍遥的技能配置和表现。

## 需求内容

### 1. 技能调整

- **普通1**：御剑术 → 飞剑术（直线剑气）
- **普通2**：四方剑阵 → 御剑术（自动寻找最近目标，可寻路追踪）
- **大招**：万剑护体（保留）

### 2. 御剑术表现

- 在飞剑术基础上
- 颜色变为金色剑
- 自动寻找距离最近的目标
- 可寻路追踪敌人

### 3. 肉鸽卡牌更新

**飞剑术（普通1）**
- 锋芒毕露 (+20%伤害)
- 分光化影 (+1发射)
- 定身咒 (定身1秒)
- 万剑齐发 (散射3道)

**御剑术（普通2）**
- 剑荡八荒 (+30%伤害)
- 剑影追踪 (+追踪距离)
- 迟滞之阵 (减速30%)
- 穿心一箭 (+穿透)

**万剑护体（大招）**
- 剑锋凌厉 (+30%伤害)
- 剑域扩张 (+30px)
- 剑鸣不绝 (+5秒)
- 剑气回天 (回血1%/秒)

# 需求文档_v1.8.0

## 商店系统

### 商店入口

- **入口位置**：主界面底部中央"商店"按钮
- **进入方式**：点击按钮弹出商店界面
- **商店分类**：
  - 钻石商店（消耗钻石购买道具）
  - 金币商店（消耗金币购买道具）

### 卡牌商店

- **刷新机制**：每6小时自动刷新1次
- **手动刷新**：消耗10钻石立即刷新
- **卡牌数量**：每次显示6张卡牌
- **卡牌来源**：随机从卡牌库抽取
- **购买限制**：每张卡牌只能购买1次

### 商店UI

- **界面样式**：全屏弹窗
- **顶部标签**：钻石商店/金币商店/卡牌商店/道具商店（左右滑动切换）
- **商品卡片**：2列排列，每张卡片显示商品图标、名称、价格
- **购买按钮**：商品卡片右下角"购买"按钮
- **关闭按钮**：右上角X按钮

### 购买确认

- 点击购买后弹出确认框
- 显示商品内容和价格
- 确认后扣除资源并发放商品
- 资源不足时按钮显示为灰色不可点击

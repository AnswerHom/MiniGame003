# 需求文档_v2.2.0

## 李逍遥技能表现优化

### 技能形态调整

所有技能表现形态从"子弹"改为"剑"，更符合仙侠风格

### 御剑术

- **形态**：飞剑
- **表现**：一道银色飞剑向前方飞行
- **颜色**：银白色带蓝光
- **尺寸**：长度30px，宽度8px
- **飞行轨迹**：直线飞行，带剑光拖尾效果

### 四方剑阵

- **形态**：四把飞剑
- **表现**：四把飞剑向四个方向（上下左右）飞出
- **颜色**：银白色带蓝光
- **尺寸**：每把飞剑长度25px，宽度6px
- **飞行轨迹**：直线飞行，带剑光拖尾效果

### 万剑护体

- **形态**：环绕飞剑
- **表现**：8把飞剑环绕角色旋转
- **颜色**：银白色带金光
- **尺寸**：每把飞剑长度20px，宽度5px
- **旋转效果**：顺时针旋转，60°/秒

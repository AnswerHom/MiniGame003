# 需求文档_v2.2.0

## 天赋系统

### 天赋入口

- **入口位置**：主界面"天赋"按钮
- **进入方式**：点击按钮弹出天赋界面
- **前置条件**：通关第5波解锁天赋系统

### 天赋树结构

#### 攻击系天赋

| 天赋名称 | 效果 | 前置 | 点数 |
|----------|------|------|------|
| 锋利 | 攻击力+5% | 无 | 1 |
| 穿透 | 攻击穿透+10% | 锋利 | 2 |
| 致命 | 暴击率+5% | 无 | 1 |
| 毁灭 | 暴击伤害+15% | 致命 | 2 |
| 连击 | 攻击速度+8% | 锋利 | 2 |
| 嗜血 | 击杀回复5%生命 | 连击 | 3 |

#### 防御系天赋

| 天赋名称 | 效果 | 前置 | 点数 |
|----------|------|------|------|
| 坚韧 | 生命值+10% | 无 | 1 |
| 铁壁 | 受到的伤害-5% | 坚韧 | 2 |
| 闪避 | 闪避率+5% | 无 | 1 |
| 轻盈 | 移动速度+5% | 闪避 | 2 |
| 再生 | 每5秒回复1%生命 | 坚韧 | 2 |
| 不死 | 死亡时满血复活1次 | 再生 | 3 |

#### 辅助系天赋

| 天赋名称 | 效果 | 前置 | 点数 |
|----------|------|------|------|
| 治疗 | 治疗效果+10% | 无 | 1 |
| 护盾 | 护盾效果+15% | 治疗 | 2 |
| 幸运 | 掉宝率+10% | 无 | 1 |
| 商人 | 商店价格-10% | 幸运 | 2 |
| 经验 | 经验获取+15% | 幸运 | 2 |
| 聚宝 | 金币获取+15% | 商人 | 3 |

### 天赋点数

- **获取方式**：每通关1波获得1点天赋点数
- **上限**：共30点天赋点数
- **重置**：消耗50钻石重置所有天赋

### 天赋UI

- **界面样式**：三列布局（攻击/防御/辅助）
- **天赋节点**：圆形节点，未点亮显示灰色，点亮显示对应颜色
- **连线显示**：已点亮天赋之间显示连接线
- **点数显示**：顶部显示可用点数和已用点数
- **保存按钮**：底部"保存"按钮确认天赋配置

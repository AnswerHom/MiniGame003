# 李逍遥万剑诀替换

## 需求概述

将李逍遥大招"万剑护体"替换为"万剑诀"。

## 需求内容

### 1. 技能替换

- 移除原有技能：万剑护体
- 新增技能：万剑诀

### 2. 万剑诀技能设计

**基础属性**
| 属性 | 数值 |
|------|------|
| 飞剑数量 | 5把 |
| 攻击范围 | 半径150px |
| 伤害 | 200%攻击力 |
| 冷却 | 60秒 |

**释放方式**
- 5把剑从角色周围升起
- 斜向下砸向指定区域（角色面向方向的前方）
- 砸中地面后造成范围伤害

### 3. 肉鸽卡牌（4张）

- 万剑齐发（+3把飞剑，共8把）
- 剑雨滂沱（+50%范围）
- 锋芒毕露（+30%伤害）
- 震颤大地（减速30%，持续3秒）

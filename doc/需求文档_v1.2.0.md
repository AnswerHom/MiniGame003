# 需求文档_v1.2.0

## 李逍遥（剑客）角色

### 角色基础属性

| 属性 | 数值 |
|------|------|
| 生命值 | 800 |
| 攻击力 | 120 |
| 攻击速度 | 1.0 |
| 移动速度 | 85 |
| 攻击范围 | 80px |
| 暴击率 | 10% |
| 暴击伤害 | 150% |

### 角色外观

- 基础形态：圆形角色，直径40px
- 头像：圆形位于角色中央，直径30px
- 血量环：圆形进度条环绕角色外圈，宽度3px
- 血量颜色：
  - 100%-51%：绿色 (#00FF00)
  - 50%-21%：黄色 (#FFFF00)
  - 20%-0%：红色 (#FF0000)

## 技能系统

### 技能1：御剑术

- **类型**：主动攻击技能
- **效果**：向前方发射一道直线剑气
- **关键参数**：
  - 伤害：150%攻击力
  - 射程：200px
  - 宽度：20px
  - 飞行速度：400px/秒
  - 冷却时间：1.5秒
- **按键**：数字键1

### 技能2：四方剑阵

- **类型**：主动攻击技能
- **效果**：向四个方向（上下左右）发射四道剑气
- **关键参数**：
  - 单道伤害：80%攻击力
  - 剑数：4道
  - 射程：120px
  - 宽度：15px
  - 飞行速度：350px/秒
  - 冷却时间：3秒
- **按键**：数字键2

### 技能3：万剑护体

- **类型**：主动大招
- **效果**：剑环绕旋转，敌靠近受伤
- **关键参数**：
  - 伤害：50%攻击力/秒
  - 环绕半径：100px
  - 持续时间：10秒
  - 剑数量：8把
  - 旋转速度：60°/秒
  - 冷却时间：30秒
- **按键**：数字键3

---

## 技能通用规则

- 技能图标：30x30px，正方形底板
- 冷却显示：技能图标上覆盖灰色遮罩，遮罩高度随冷却时间减少
- 快捷键：1/2/3分别对应技能1/2/3
- 施放方式：按下对应数字键即可释放
- 技能释放方向：自动朝向当前面向方向或最近敌人

---

## 战斗系统

### 普通攻击

- 攻击方式：近战挥砍
- 攻击动画：角色向前挥动武器
- 攻击间隔：1/攻击速度 = 1秒
- 攻击范围：80px

### 暴击判定

- 暴击伤害：150%普通伤害
- 暴击特效：攻击时显示金色闪光

---

## 移动控制

- 虚拟摇杆：屏幕左侧区域
- 摇杆大小：100px直径
- 摇杆样式：半透明圆形
- 移动响应：平滑移动

# 需求文档_v1.4.0

## 波次系统

### 波次基本规则

- **波次间隔**：每波次间隔15秒
- **波次提示**：倒计时3秒时显示"下一波即将来袭"
- **波次提示位置**：屏幕顶部居中
- **提示样式**：红色文字，持续显示直到波次开始

### 波次难度递增

- **波次1**：小怪数量5只，基础属性100%
- **波次2**：小怪数量6只，基础属性110%
- **波次3**：小怪数量7只，基础属性120%
- **后续每波**：小怪数量+1，属性+10%
- **精英怪**：从第3波开始，每3波出现1只精英怪

### 怪物类型

#### 普通小怪

- **生命值**：100
- **攻击力**：20
- **移动速度**：50px/秒
- **体积**：直径20px
- **颜色**：灰色
- **行为**：自动追踪最近的角色

#### 精英怪

- **生命值**：300
- **攻击力**：40
- **移动速度**：40px/秒
- **体积**：直径30px
- **颜色**：紫色
- **特殊效果**：周围小怪移动速度+20%
- **行为**：优先攻击血量最低的角色

### 波次结算

- **每波奖励**：每完成一波，发放100金币
- **波次显示**：屏幕左上角显示"第X波"
- **完成条件**：击杀所有怪物

### 刷怪规则

- **刷怪位置**：屏幕边缘随机位置
- **刷怪间隔**：每0.5秒刷1只，直到达到本波数量
- **刷怪范围**：屏幕外100px缓冲区

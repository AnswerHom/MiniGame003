# 需求文档_v2.3.3

## 小怪数值设计

### 战斗波次

- **总波次**：12波 / 1场副本
- **第12波**：BOSS战

### 每波怪物数量（割草爽感）

| 波次 | 怪物数量 | 难度 |
|------|----------|------|
| 1-3 | 15-20只 | 热身 |
| 4-6 | 25-30只 | 开始爽 |
| 7-9 | 35-45只 | 小高潮 |
| 10-11 | 50-60只 | 爽感巅峰 |
| 12 | 40-50只 + BOSS | 最终挑战 |

### 数值成长（每波）

- **生命值成长**：每波+8%
- **攻击力成长**：每波+6%
- **移动速度成长**：每波+1%（上限+30%）

- 攻击力 × 2

# 需求文档_v2.1.0

## 装备系统

### 装备类型

| 部位 | 名称 | 基础属性 |
|------|------|----------|
| 武器 | 剑/法杖 | 攻击力+20~50 |
| 防具 | 衣服/袍 | 生命值+100~300 |
| 饰品 | 项链/戒指 | 暴击率+5%~15% |

### 装备品质

- **普通（灰色）**：基础属性
- **优秀（绿色）**：基础属性+10%
- **精良（蓝色）**：基础属性+20%
- **史诗（紫色）**：基础属性+30%
- **传说（橙色）**：基础属性+50%

### 装备获取

- **BOSS掉落**：BOSS战概率掉落装备
- **商店购买**：钻石商店随机刷新装备
- **抽卡赠送**：抽卡时概率附带装备

### 装备强化

- **强化消耗**：金币
- **强化等级**：+1~+10
- **强化成功率**：100%/90%/80%...每级降低10%
- **强化效果**：每级+5%属性
- **失败惩罚**：强化等级降1级

### 装备精炼

- **精炼消耗**：相同装备或精炼石
- **精炼等级**：0~5
- **精炼效果**：每级+10%基础属性

### 装备UI

- **入口**：主界面"装备"按钮
- **列表**：显示已拥有的装备
- **详情**：点击装备显示属性详情
- **强化按钮**：装备详情页强化入口
- **精炼按钮**：装备详情页精炼入口

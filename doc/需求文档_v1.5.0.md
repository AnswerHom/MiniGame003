# 需求文档_v1.5.0

## 抽卡选角系统

### 抽卡界面

- **入口**：游戏开始时显示"开始游戏"按钮，点击后进入抽卡界面
- **抽卡方式**：十连抽
- **展示**：10张角色卡牌同时翻开
- **卡牌数量**：3个角色（李逍遥、赵灵儿、阿奴）循环出现

### 卡牌样式

- **卡牌尺寸**：80px × 100px
- **卡牌背景**：角色专属颜色
  - 李逍遥：蓝色
  - 赵灵儿：粉色
  - 阿奴：紫色
- **角色头像**：卡牌中央显示角色头像（30px圆形）
- **角色名称**：卡牌底部显示角色名称
- **星级**：卡牌右上角显示星级（★）
- **选中状态**：金色边框高亮

### 抽卡规则

- **保底机制**：每10抽必出当前未拥有的角色
- **初始赠送**：游戏开始赠送200钻石
- **抽卡消耗**：20钻石/抽（单抽）或 180钻石/10连抽
- **重复角色**：重复获得角色时，给予50钻石补偿

### 队伍配置

- **队伍人数**：最多5人
- **初始角色**：系统赠送1个随机角色
- **上阵条件**：抽到角色后自动上阵
- **替换功能**：点击已上阵角色可查看详情，长按可下阵

### 钻石系统

- **钻石获取**：
  - 初始：200钻石
  - 波次奖励：每波100金币可兑换10钻石（汇率10:1）
  - 重复角色：50钻石/次
- **钻石显示**：界面右上角显示当前钻石数量

### 金币系统

- **金币获取**：波次完成后获得100金币
- **金币显示**：界面右上角钻石旁边显示金币数量
- **兑换功能**：金币可兑换钻石（10:1）

### UI布局

- **顶部**：钻石数量（右上角）、金币数量（右上角）
- **中部**：抽卡区域（10张卡牌3-4-3排列）
- **底部**：单抽按钮（20钻）、十连抽按钮（180钻）

# 需求文档_v1.9.0

## 战斗地图

### 地图概述

- **类型**：俯视视角战斗场景
- **风格**：仙侠古风，水墨山水
- **区域**：仙剑岛、锁妖塔、神木林等

### 地图元素

- **地面**：青石板/草地纹理
- **边界**：山脉/竹林/仙雾
- **战斗区域**：圆形/六边形场地，淡金色边框
- **装饰**：飘落花瓣、萤火虫

### 刷怪规则

- 怪物从地图边缘进入
- 每波次间隔15秒
- 难度递增：数量+1，属性+10%/波

### 波次系统

- 波次显示：屏幕左上角"第X波"
- 波次提示：倒计时3秒显示"下一波即将来袭"
- 波次奖励：每波100金币

### 战斗区域

- 角色死亡后显示倒地状态
- 全部阵亡 = 游戏结束
- 虚拟摇杆控制移动

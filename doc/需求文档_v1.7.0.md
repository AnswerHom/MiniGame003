# 需求文档_v1.7.0

## 肉鸽随机事件系统

### 事件触发时机

- **波次间隙**：每波次结束后（战斗结算前）触发
- **触发概率**：60%概率触发随机事件
- **事件数量**：每次随机事件从事件库中抽取3个，玩家选择1个执行

### 事件类型

#### 增益类事件

| 事件名称 | 效果 | 持续时间 |
|----------|------|----------|
| 仙灵祝福 | 全队攻击力+15% | 本波 |
| 体力充沛 | 全队移动速度+10% | 本波 |
| 铜墙铁壁 | 全队受到的伤害-15% | 本波 |
| 鸿运当头 | 暴击率+10% | 本波 |
| 灵泉涌现 | 全队每秒回复1%生命 | 本波 |

#### 资源类事件

| 事件名称 | 效果 |
|----------|------|
| 宝箱奖励 | 获得50钻石 |
| 金币大礼包 | 获得200金币 |
| 神秘商人 | 消耗100金币抽取1张随机卡牌 |

#### 挑战类事件

| 事件名称 | 效果 |
|----------|------|
| 精英怪突袭 | 立即刷出1只精英怪，击杀后获得100金币 |
| 限时挑战 | 30秒内击杀10只怪，奖励100钻石 |
| 生命考验 | 扣除全队20%当前生命，获得50钻石 |

#### 特殊类事件

| 事件名称 | 效果 |
|----------|------|
| 角色召唤 | 随机召唤1个已有角色（不重复） |
| 技能强化 | 随机强化1个技能的冷却时间-20% |
| 休息整顿 | 清除所有负面状态，全队回满血 |

### 事件UI

- **界面样式**：弹窗式居中显示
- **背景**：半透明黑色遮罩
- **事件卡片**：3张横向排列，每张180x120px
- **卡片内容**：事件图标（顶部）+ 事件名称（中部）+ 事件描述（底部）
- **选择方式**：点击任意卡片触发事件
- **跳过选项**：如不选择，5秒后自动跳过本轮事件

### 事件权重

- 增益类：40%
- 资源类：25%
- 挑战类：20%
- 特殊类：15%

# 需求文档_v1.7.0

## 游戏大厅

### 入口与流程

- 玩家进入游戏后首先显示游戏大厅
- 大厅是游戏的主界面

### UI布局

| 区域 | 内容 |
|------|------|
| 顶部 | 游戏标题 + 玩家信息（等级、钻石、金币） |
| 中部 | 角色展示区（已拥有的角色卡牌） |
| 底部 | 功能按钮（抽卡、开始战斗等） |

### 功能区域

#### 抽卡入口
- 位置：大厅中央
- 按钮文字："抽卡"
- 点击后弹出抽卡界面

#### 角色选择
- 展示已拥有的角色卡牌
- 点击角色卡牌可查看详情
- 可装备/卸下卡牌

#### 队伍配置
- 选择最多5个角色进入战斗
- 点击角色卡牌勾选/取消勾选
- 已选中的角色显示勾选标记

#### 开始战斗按钮
- 位置：大厅底部
- 按钮文字："开始战斗"
- 点击条件：至少选择1个角色
- 点击后进入战斗地图

### 资源显示

- 钻石：右上角显示数量
- 金币：右上角显示数量

### 抽卡界面

- 抽卡方式：十连抽
- 展示：10张角色卡牌同时翻开
- 卡牌数量：3个角色（李逍遥、赵灵儿、阿奴）
- 保底机制：每10抽必出当前未拥有的角色
- 初始赠送：200钻石
- 抽卡消耗：20钻石/抽 或 180钻石/10连抽

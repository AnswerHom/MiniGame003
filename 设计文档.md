# 仙剑肉鸽 - 设计文档

---

## 项目定位

- **类型**：肉鸽割草 + 组队生存
- **视角**：俯视上帝视角
- **平台**：手机端（竖屏）
- **特色**：仙剑IP角色 + 肉鸽卡牌Build

---

## 核心玩法

1. 5人角色组队协同战斗
2. 多波次怪物进攻，生存挑战
3. 角色全部阵亡 = 游戏结束
4. 大地图探索，虚拟摇杆控制移动
5. 抽卡选角 + 肉鸽随机事件

---

## 角色定位

剑客、巫师、药师

---

## 角色属性（7个）

生命值、攻击力、攻击速度、移动速度、攻击范围、暴击率、暴击伤害

---

## 李逍遥（剑客）

### 属性
| 属性 | 数值 |
|------|------|
| 生命值 | 800 |
| 攻击力 | 120 |
| 攻击速度 | 1.0 |
| 移动速度 | 85 |
| 攻击范围 | 80px |
| 暴击率 | 10% |
| 暴击伤害 | 150% |

### 技能
| 类型 | 名称 | 效果 | 冷却 |
|------|------|------|------|
| 普通 | 御剑术 | 单体剑气，150%伤害 | 1.5秒 |
| 普通 | 四方剑阵 | 4道剑气向四周发射，320%伤害 | 3秒 |
| 大招 | 万剑护体 | 剑环绕旋转，敌靠近受伤，持续10秒 | 30秒 |

### 肉鸽卡牌（12张）

**御剑术**
- 锋芒毕露：伤害+30%
- 分光化影：分裂2把子剑，子剑30%伤害
- 定身咒：眩晕1秒
- 万剑齐发：同时发射2把剑

**四方剑阵**
- 剑荡八荒：剑数+4
- 剑阵加强：伤害+20%
- 迟滞之阵：减速30%
- 幻剑生花：剑数+2

**万剑护体**
- 剑锋凌厉：伤害+25%
- 剑域扩张：范围+50%
- 剑鸣不绝：时间+5秒
- 剑气回天：每秒恢复1%生命

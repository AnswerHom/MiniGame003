## 角色系统

### 李逍遥（剑客）
- 技能：御剑术、四方剑阵、万剑护体
- 特点：物理攻击型
- 御剑术：前方直线剑气（150%攻击、200px射程、1.5秒冷却）
- 四方剑阵：四向剑气（80%×4攻击、120px射程、3秒冷却）
- 万剑护体：环绕剑阵（50%攻击/秒、100px范围、持续10秒、30秒冷却）

### 赵灵儿（药师）
- 技能：五雷咒、观音咒、圣灵复活
- 特点：治疗辅助型
- 五雷咒：治疗范围内队友（80%攻击、150px范围、2秒冷却）
- 观音咒：为队友套护盾（20%生命、100px范围、4秒冷却）
- 圣灵复活：复活死亡队友（1人、30秒冷却）

## 技能系统

### 快捷键
- 1：使用第一个技能
- 2：使用第二个技能
- 3：使用第三个技能

### 技能类型
- attack：攻击技能
- aoe：范围攻击（万剑护体）
- heal：治疗技能
- shield：护盾技能
- revive：复活技能

## 角色外观

### 血量环
- 100%-51%：绿色 (#00FF00)
- 50%-21%：黄色 (#FFFF00)
- 20%-0%：红色 (#FF0000)

## 移动控制

### 虚拟摇杆
- 屏幕左侧区域
- 100px直径半透明圆形
- 拖动控制移动方向

### 鼠标/键盘
- 鼠标点击：移动角色
- 数字键1/2/3：使用技能

## 暴击系统

- 暴击率：10%（李逍遥）
- 暴击伤害：150%
- 暴击特效：金色闪光环绕投射物

## 肉鸽卡池

### 李逍遥卡池
- 御剑术强化：锋芒毕露、分光化影、定身咒、万剑齐发
- 四方剑阵强化：剑荡八荒、剑阵加强、迟滞之阵、幻剑生花
- 万剑护体强化：剑锋凌厉、剑域扩张、剑鸣不绝、剑气回天

### 赵灵儿卡池
- 五雷咒强化：雷光普照、群体治愈、持续治疗、雷劫
- 观音咒强化：金刚护体、护盾强化、反射护盾、水灵护盾
- 圣灵复活强化：复活祝福、群体复活、快速复活、复活传承

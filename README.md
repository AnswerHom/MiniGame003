
## v2.3.1 更新 - 队伍系统

### 队伍配置
- 最大人数：5人
- 最少人数：1人
- 入口：游戏大厅→队伍配置按钮

### 队形系统
- 横排：角色一字排开
- 纵列：角色排队跟随（默认）
- 切换方式：战斗前在大厅切换

### 战斗中的队伍行为
- 跟随移动：第2-5号角色跟随第1号角色移动
- 跟随距离：相邻角色间距30px
- 转向同步：所有角色同步转向

## 角色系统

### 李逍遥（剑客）
- 技能：御剑术、四方剑阵、万剑护体
- 特点：物理攻击型
- 御剑术：前方直线剑气（150%攻击、200px射程、1.5秒冷却）
- 四方剑阵：四向剑气（80%×4攻击、120px射程、3秒冷却）
- 万剑护体：环绕剑阵（50%攻击/秒、100px范围、持续10秒、30秒冷却）

### 赵灵儿（药师）
- 技能：五雷咒、观音咒、圣灵复活
- 特点：治疗辅助型
- 五雷咒：治疗范围内队友（80%攻击、150px范围、2秒冷却）
- 观音咒：为队友套护盾（20%生命、100px范围、4秒冷却）
- 圣灵复活：复活死亡队友（1人、30秒冷却）

### 阿奴（巫师）
- 技能：风雪冰天、雷劫、阴阳逆转
- 特点：魔法攻击型
- 风雪冰天：扇形冰锥（120%攻击、60°扇形、150px射程、2秒冷却、带减速）
- 雷劫：区域雷电（180%攻击、60px半径、3秒冷却、延迟0.3秒降落）
- 阴阳逆转：击退敌人（100%攻击、150px范围、25秒冷却）

## 肉鸽卡牌系统

### 卡牌类型
- 攻击类：御剑术、四方剑阵、万剑护体相关卡牌
- 功能类：五雷咒、观音咒、圣灵复活相关卡牌

### 卡牌稀有度

- 传说（橙色）：顶级属性加成或强力特殊效果

- 每个角色最多装备3张卡牌

## BOSS战系统

### BOSS触发
- 每5波触发1次BOSS战
- 第5波：毒娘子，第10波：赤鬼王，第15/20波+：拜月教主

### BOSS类型
- 毒娘子：5000HP，扇形毒液攻击，召唤小怪，狂暴状态
- 赤鬼王：8000HP，地面冲击/岩石投掷，免疫正面伤害
- 拜月教主：15000HP，魔法弹/暗黑漩涡/雷电，无敌时间

### BOSS战斗机制
- 弱点系统
- 无敌/狂暴状态
- 团队配合
- 伤害统计

### BOSS掉落
- 首杀奖励：500钻石
- 常规掉落：200-500金币，1-2张卡牌

## 装备系统

### 装备类型
- 武器：剑/仙剑/魔剑/圣剑/法杖/魔杖/贤杖
- 防具：布衣/皮甲/铁甲/法袍/圣袍/天衣
- 饰品：项链/戒指/吊坠/耳环/手镯

### 装备品质


## 角色系统

### 李逍遥（剑客）
- 技能：御剑术、四方剑阵、万剑护体
- 特点：物理攻击型

### 赵灵儿（药师）
- 技能：五雷咒、观音咒、圣灵复活
- 特点：治疗辅助型
- 五雷咒：治疗范围内队友（治疗80%攻击、范围150px、冷却2秒）
- 观音咒：为队友套护盾（护盾20%生命、范围100px、冷却4秒）
- 圣灵复活：复活死亡队友（复活1人、30秒冷却）

## 技能系统

### 快捷键
- 1：使用第一个技能
- 2：使用第二个技能
- 3：使用第三个技能

### 技能类型
- attack：攻击技能
- aoe：范围攻击
- heal：治疗技能
- shield：护盾技能
- revive：复活技能

## 肉鸽卡池

### 李逍遥卡池
- 御剑术强化：锋芒毕露、分光化影、定身咒、万剑齐发
- 四方剑阵强化：剑荡八荒、剑阵加强、迟滞之阵、幻剑生花
- 万剑护体强化：剑锋凌厉、剑域扩张、剑鸣不绝、剑气回天

### 赵灵儿卡池
- 五雷咒强化：雷光普照、群体治愈、持续治疗、雷劫
- 观音咒强化：金刚护体、护盾强化、反射护盾、水灵护盾
- 圣灵复活强化：复活祝福、群体复活、快速复活、复活传承

## 游戏操作

- 鼠标点击：移动角色
- 数字键1/2/3：使用对应技能

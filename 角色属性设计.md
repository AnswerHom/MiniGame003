# 仙剑肉鸽 - 角色属性设计

---

## 角色属性（7个）

| 属性 | 说明 |
|------|------|
| 生命值 | 角色HP上限 |
| 攻击力 | 每次攻击伤害 |
| 攻击速度 | 每秒攻击次数 |
| 移动速度 | 移动快慢 |
| 攻击范围 | 可攻击距离 |
| 暴击率 | 暴击概率 |
| 暴击伤害 | 暴击额外伤害 |
